# zon2tct

This is a Code 2 creation tool for the webgame *The Campaign Trail*, and its forks, such as *[Campaign Trail Showcase](https://campaigntrailshowcase.com)*, *[New Campaign Trail](https://www.newcampaigntrail.com)*, and other forks.

It allows for creating mods by writing the questions, answers, and the answer feedback and effects of those answers, in a single file with a hierarchical data structure. This tool then reads the input file and outputs a JavaScript file with the hierarchical format converted to the relational format needed for *The Campaign Trail*.

## How to use

Download the archive for your operating system (i.e. if on Windows 64-bit, download `zon2tct-x86_64-windows.zip` or `zon2tct-x86_64-windows.tar.gz`). Each archive contains a single executable file. To use it, invoke the executable file with two arguments — the first argument is the input file, written in ZON, and the second argument is the name of the output file.
